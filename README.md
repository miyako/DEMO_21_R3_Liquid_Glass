# DEMO_21_R3_Liquid_Glass

## Installing and Using a 4D Project

This demo project showcases the new visual capabilities introduced in 4D 21 R3 with modern interfaces using Liquid Glass on macOS and Fluent UI on Windows.

The application demonstrates how list and detail forms automatically adopt the native style of the operating system without requiring any specific code changes or form redesign.

The demo also highlights the new printing engine in 4D 21 R3, allowing list and detail forms to be printed with a cleaner and more readable paper-friendly rendering while preserving the modern on-screen appearance of the interface.

Find more information on the [4D blog](https://blog.4d.com/):
* [Liquid Glass on macOS Tahoe](https://blog.4d.com/the-new-macos-tahoe-design-comes-to-your-4d-applications/)
* [a dedicated print rendering](https://blog.4d.com/printing-modern-interfaces-with-clean-consistent-output/)
* [the object library](https://blog.4d.com/a-modernized-object-library-with-fluent-ui/)

### Pre-requisites

* Download the latest Release version of 4D from: https://us.4d.com/product-download or the latest Beta version from: https://discuss.4d.com
* Follow the activation steps for 4D from: https://developer.4d.com/docs/GettingStarted/installation

### Steps to Run the Project

* Clone or download the GitHub repository containing the 4D project to your local machine. Need help, check out [this blog](https://blog.4d.com/github-4d-depot/).
* Open the 4D project in your 4D software by navigating to "File > Open Project".  You can find more details [here](https://developer.4d.com/docs/GettingStarted/creating#opening-a-project).
* Play with this HDI.
* Navigate to the "Mode/Return to design mode" menu to view the code.

By following these steps, you will be able to successfully install and run a 4D project.
